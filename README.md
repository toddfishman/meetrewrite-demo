# Rewrite — Interactive Workflow Demo

An interactive, self-contained demo of the [Rewrite](https://meetrewrite.com) product workflow:
turning a website / web app / storefront into a **native iOS app** (Swift / SwiftUI) — *preview free, pay when it ships*.

Walk three sample projects (a Shopify storefront, a React SaaS app, a Next.js tool) through the real
**13-step, 3-phase workflow**: scan & scope → preview & quote → build & handoff.

- Step through manually, click any step in the rail, or hit **Auto-run** to watch the whole thing.
- Includes a guided narration tour for first-time viewers.

Single static `index.html`, no backend. Brand styling (navy + pink→purple→cyan gradient, Space Grotesk) matched to meetrewrite.com.

*Unofficial demo — all projects, scans, quotes and screens are fictional examples.*
